## Usage

### Inputs

`../data/glucose_data.csv`

CSV file downloaded from <https://www.libreview.com/>.

`../data/date_annotations.csv`

CSV file with two (unnamed) columns:

1. Date in the format `DD-MM-YYYY`.
2. Custom labels used to group days.

Example

```
17-05-2025,travel
18-05-2025,weekend
```
